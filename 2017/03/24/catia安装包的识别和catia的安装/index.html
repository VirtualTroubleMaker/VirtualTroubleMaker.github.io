catia安装包的识别和catia的安装

发表于 2017-03-24

catia v5 p1 p2 p3的区别

“p”——platform是平台的意思，指的是工作中设置的工作环境和界面，由p1到p3依次由低到高，适用于不同的工作环境和行业，一般人员用p2，p3主要面对高端和专业用户，对机子的要求也最高。

P1、p2和p3应用平台都是在相同的数据模型中操作，并使用相同的设计方法，具有相同的用户界面。P3包含P2的所有功能，P2包含P1的所有功能。

一个安装包的详细说明

其实软件公司的版本号都是有个标准的。以DS CATIA P3 V5-6 R2016 GA WIN64-SSQ为例： - DS：达索公司（Dassault Systemes） - CATIA：Computer Aided Three-Dimensional Interface Application - V5-6 R2016：软件版本 - GA：General Availability，就是公开发售的发行版，不带service pack。 - P3：功能版本，区别于P1，P2，P3最高级 - SSQ：SolidSQUAD（破解组织） - V(version)相当于系列号(或总版本号,有算法等根本上的改动才会用这个)，就如： V4，V5，代表catia的大版本。 - R(Release)相当于版本号 如：R9~R17。代表catia不同大版本下的小的修订版。 - SP是针对不同版本打出的补丁如SP1,SP2(Service Pack) - HF，hotfix，热补丁或者热修复

DS CATIA P3 V5-6 R2016 GA WIN64-SSQ 这个版本只有SP1和SP2。现在还没有HF。

问题：

1. CATIA V5-6R2014 全新安装到SP5,需要从SP1一直安装到SP5吗？

SP的版本只需要安装最新的就可以了，HF版本需要先安装对应的SP版本。

安装

  1. 先安装catia本体，安装完可以在开始菜单里catia目录下找到“软件管理”
  2. 安装文档（可选中英文）
  3. 破解catia本体
  4. 安装SP，可以在“软件管理”中提交SP，需要再次破解，利用SP包的破解dll
  5. 安装HF，无需破解，会发现安装SP和HF后，“软件管理”均有变化
